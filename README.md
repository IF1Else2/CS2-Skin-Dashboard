# CS2 Skin Market Dashboard

Dashboard interactivo de skins de CS2/CSGO hecho con Python, Plotly e ipywidgets.

## Instalacion

```bash
pip install -r requirements.txt
```

## Ejecutar

Abrir el notebook:

```bash
jupyter notebook
```

o ejecutar desde VSCode.

## Librerias usadas

- pandas
- numpy
- plotly
- ipywidgets
- requests
